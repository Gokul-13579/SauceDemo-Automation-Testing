SauceDemo Automation Testing Project
Project Overview

This repository contains automation test scripts for the SauceDemo web application using Selenium WebDriver with Java and TestNG.
The project demonstrates automated validation of key application functionalities using a structured test automation framework.

Application Under Test

https://www.saucedemo.com/

Tools & Technologies

* Java
* Selenium WebDriver
* TestNG
* Maven

Framework Design

Page Object Model (POM)

Automated Test Scenarios

* Verify login functionality with valid credentials
* Verify login with invalid credentials
* Verify product listing page
* Verify add to cart functionality
* Verify cart validation
* Verify logout functionality

Project Structure

src/test/java
├── base
├── pages
├── tests

How to Run the Tests

* Clone the repository
* Open the project in an IDE (Eclipse / IntelliJ)
* Install Maven dependencies
* Run the testng.xml file to execute the test suite
